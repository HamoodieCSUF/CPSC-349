CPSC 349 - Movie Explorer Application
=====================================

IMPORTANT - API KEY SETUP:
Before using the application, you MUST get your own TMDB API key:

1. Go to https://www.themoviedb.org/
2. Create a free account
3. Go to Settings > API and request an API key
4. Replace the API_KEY in script.js (line 2) with your own key

FEATURES IMPLEMENTED:
✅ Movie Display: Shows 20 movies per page with poster, title, release date, and rating
✅ Pagination: Previous/Next buttons with current page display
✅ Search: Search movies by title using the search bar
✅ Sorting: Sort by popularity, release date, and rating
✅ Responsive Design: Works on desktop, tablet, and mobile
✅ Error Handling: Shows appropriate messages for API failures
✅ Loading States: Loading indicator while fetching data

HOW TO TEST:
1. Get your TMDB API key (see above)
2. Update the API_KEY in script.js
3. Open index.html in Google Chrome
4. Test all features:
   - Browse movies using pagination
   - Search for specific movies
   - Try different sorting options
   - Test on different screen sizes

TECHNICAL DETAILS:

- Modern ES6+ JavaScript with async/await
- Professional styling with gradients and animations
- Cross-browser compatible
